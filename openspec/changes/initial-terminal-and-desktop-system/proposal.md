# Proposal: 初始终端与上位机系统

## 背景

本项目用于实现一套基于插排外部热特征的非侵入式过载监测系统。系统由下位机监测终端和 PC 上位机组成。下位机安装在插排输入线外部，通过多点 NTC 采集导线外皮温度和环境温度，提取热特征并完成电流反演、过载判断和本地声光报警；上位机负责实时显示、参数配置、实验数据记录和后续模型调试。

该 change 的目的，是先把毕业设计的软件、硬件和通信边界固定下来，避免后续开发时功能范围反复变化。

## 目标

- 建立 ESP32-S3 下位机固件工程。
- 实现 3 路 NTC 温度采集、温度换算、滤波和热特征提取。
- 预留电流反演和轻量 AI 过载识别接口，初期允许使用阈值或占位模型跑通流程。
- 实现 LED 与蜂鸣器本地报警。
- 实现 Wi-Fi 实时数据上传和参数接收。
- 建立 PC 上位机工程，实现实时曲线、状态显示、参数配置和 CSV 保存。
- 明确上下位机 JSON 通信协议。
- 保留后续论文写作所需的硬件框图、软件流程图、协议表和实验记录入口。

## 非目标

- 不进入插排内部，不破坏插排原有结构。
- 不把电流传感器作为终端算法输入。
- 不实现 OLED 显示。
- 不实现 SD 卡本地存储。
- 第一阶段不实现复杂云平台。

## 影响范围

- `firmware/`：新增 ESP32-S3 固件工程。
- `desktop-app/`：新增 PC 上位机工程。
- `hardware/`：放置原理图、PCB、BOM 和测温夹具资料。
- `docs/`：维护系统方案、通信协议、测试记录和论文素材。
- `models/`：后续放置数据处理脚本、训练脚本、模型文件和量化结果。
- `data/`：后续放置采集数据和标定数据。
