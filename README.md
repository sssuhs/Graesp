# GraEsp

插排外置式过载监测终端与上位机项目。

本项目用于毕业设计代码与硬件资料管理，系统由下位机监测终端和 PC 上位机组成。下位机基于 ESP32-S3-WROOM-1-N16R8，通过外置 NTC 热敏电阻采集插排输入线外皮温度和环境温度，完成温度滤波、热特征提取、电流反演、边缘 AI 过载判别与声光报警；上位机通过 Wi-Fi 接收数据，完成实时曲线显示、参数配置和实验数据保存。

## 目录

- `firmware/`：ESP32-S3 下位机固件。
- `desktop-app/`：PC 上位机软件。
- `hardware/`：PCB、原理图、BOM、外壳和夹具设计资料。
- `docs/`：项目方案、通信协议、需求和论文素材。
- `data/`：实验采集数据。
- `models/`：模型训练、量化和部署文件。
- `openspec/`：OpenSpec 规格与变更记录。

## 设计边界

- 不拆开插排，不进入插排内部。
- 不将电流传感器作为终端算法输入。
- 终端仅基于外部热特征进行电流反演和过载预测。
- 上位机负责显示、记录和参数配置；终端断开上位机后仍应能独立报警。
